# SDK Changes

This page documents detailed changes to the Laravel eBay SDK package.

## Version 0.2.0 (Alpha)

- Support for **Account API v2.2.0**
    - `Rat\eBaySDK\API\AccountAPIv2\CombinedShippingRules\CreateCalculatedShippingRules`
    - `Rat\eBaySDK\API\AccountAPIv2\CombinedShippingRules\CreateFlatShippingRules`
    - `Rat\eBaySDK\API\AccountAPIv2\CombinedShippingRules\CreatePromotionalShippingRule`
    - `Rat\eBaySDK\API\AccountAPIv2\CombinedShippingRules\GetCombinedShippingRules`
    - `Rat\eBaySDK\API\AccountAPIv2\CombinedShippingRules\UpdateCalculatedShippingRules`
    - `Rat\eBaySDK\API\AccountAPIv2\CombinedShippingRules\UpdateCombinedPayments`
    - `Rat\eBaySDK\API\AccountAPIv2\CombinedShippingRules\UpdateFlatShippingRules`
    - `Rat\eBaySDK\API\AccountAPIv2\CombinedShippingRules\UpdatePromotionalShippingRule`
    - `Rat\eBaySDK\API\AccountAPIv2\UserPreferences\GetUserPreferences`
    - `Rat\eBaySDK\API\AccountAPIv2\UserPreferences\SetUserPreferences`
- Support for **Finances API v1.18.0**
    - `Rat\eBaySDK\API\FinancesAPI\BillingActivity\GetBillingActivities`
- Support for **Metadata API v1.13.0**
    - `Rat\eBaySDK\API\MetadataAPI\Marketplace\GetMinimumListingPricePolicies`
