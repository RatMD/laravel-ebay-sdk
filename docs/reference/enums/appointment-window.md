# AppointmentWindow Enum <DocsBadge path="sell/fulfillment/types/sel:AppointmentWindowEnum" />

An enumerated type for window slot values.

| Value     | Description                            |
| --------- | -------------------------------------- |
| `MORNING` | Appointment window starts before noon. |
| `EVENING` | Appointment window starts after noon.  |

## Example

```php
use Rat\eBaySDK\Enums\AppointmentWindow;

AppointmentWindow::MORNING;
```
