# TrafficReportDimension Enum <DocsBadge path="sell/analytics/resources/traffic_report/methods/getTrafficReport#h3-uri-parameters" />

An enum that defines the regions to which a profile can apply.

| Value     | Description                        |
| --------- | ---------------------------------- |
| `DAY`     | Generate Traffic Report by Days    |
| `LISTING` | Generate Traffic Report by Listing |

## Example

```php
use Rat\eBaySDK\Enums\TrafficReportDimension;

TrafficReportDimension::LISTING;
```
